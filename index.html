Under the Hood

How It Works

Two layers of protection work together to keep your sensitive data out of AI prompts.

01

⌨️

You Type

As you type into any AI interface — ChatGPT, Gemini, Claude — Redactron silently monitors your input in real time.

02

🧠

ML Analysis

Your prompt is sent to a fine-tuned DistilBERT model that classifies it as Safe, PII, or Sensitive based on context and intent.

03

🔍

Rule Engine

A regex-based rule layer catches structured patterns like emails, phone numbers, Aadhaar numbers, and API keys.

04

🛡️

Smart Redaction

If a risk is detected, your popup alerts you. Click Redact and sensitive parts are replaced with [EMAIL], [PHONE], [API_KEY] — so you can continue safely.

Get Started

Install in 4 Steps

No Chrome Web Store needed. Install directly in under a minute.

1

Download the Extension

Click the Download button above to get the latest redactron.zip from GitHub Releases.

2

Unzip the File

Extract the zip to a folder on your computer. Remember where you saved it.

3

Open Chrome Extensions

Go to chrome://extensions in your browser and enable Developer Mode using the toggle in the top right.

4

Load the Extension

Click Load unpacked and select the unzipped folder. Redactron will appear in your toolbar — you're protected! 🛡️
